## Installation

```bash
$ yarn install - will install all the dependencies
```

## Running the project
É necessário a API

No projeto [vsdi-api](https://github.com/wfelipedev/vsdi-api)
```bash
$ yarn start
```

Agora, no projeto vsdi-front, execute
```bash
$ yarn start
```
